Create Pattern - Fluctuation

Use this to create Fluctuation pattern.

Element Description

Exploration Name

 Click this area to expand or collapse the parameters area.

Event stream

 Select an event stream as the source for the pattern.

Tracking Fields

 Enter a value in this field. These are the fields in which you want to track the trend.
Value Field Enter a value in this field. You can select only 1 item. This field provides a value for which you want to identify a change in the trend.
Window Enter a numeric value and select its unit. This is the interval in which the application looks for a matching pattern.
Deviation Threshold% Enter a numeric value in this field. This value indicates the threshold. When the value varies beyond this threshold value, an event is generated for the pattern.
Live Output Stream This area displays the live output stream.
Charts This area displays the data in graphical format.
Return To Catalog Click this button when you want to go back to the catalog.
