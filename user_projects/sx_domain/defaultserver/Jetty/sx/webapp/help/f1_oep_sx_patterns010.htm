Create Pattern - Down Trend

Use this to create Down Trend pattern.

Element Description

Exploration Name

 Click this area to expand or collapse the parameters area.

Event stream

 Select an event stream as the source for the pattern.

Tracking Fields

 Enter a value in this field. These are the fields in which you want to track the trend.
Window Enter a numeric value and select its unit. This is the interval within which the application looks for a pattern match.
Value Field Enter a value in this field. You can select only 1 item. This is a source field whose behavior follows the trend for the pattern to emit the event.
Live Output Stream This area displays the live output stream.
Charts This area displays the data in graphical format.
Return To Catalog Click this button when you want to go back to the catalog.
