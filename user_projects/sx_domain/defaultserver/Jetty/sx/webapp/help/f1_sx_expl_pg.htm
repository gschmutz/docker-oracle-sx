Exploration Page

Use this view or edit the exploration details.

Element Description

Return to Catalog

 Click this button to navigate to the catalog.

Exploration Name

 Click this area to expand or collapse the exploration configuration area.

Sources

 Click this area to add or remove sources. After you add more than one source you must correlate them.

Add a Correlation

 Click this button to add a correlation between multiple sources to view the live output data.

Summaries

Click this area to view or modify the summaries of the exploration. An exploration can have multiple summaries.

Add a Summary

 Click this button to add a summary to the exploration. You can use this summary in the Filters.

Group By

 Click this button to select the fields by which you want to group the summary. You can move the fields up and down and rearrange the fields in the Group By menu.

Done

 Click this button when you have selected the fields by which to group the summary.

Filters

 Click this area to view or modify the filters for an exploration. The conditions with which the output can be filtered are known as Filters. When you apply filters to the output, only the events that match ALL or ANY of the filters are displayed.

Add a Filter

 Click this button to add a filter to the exploration. You can filter events by comparing a left operand with right operand using the following operators:

  • equals

  • not equals

  • lower than

  • lower than or equals

  • greater than

  • greater than or equals

  • contains.

Left operand can be any one of the following:

  • a field of any source in the exploration

  • a summary that is defined in the Summaries area.

Right operand can be any one of the following:

  • a constant of a type that is compatible with the type of left operand

  • a field or a summary of a field from a correlated source.

Match All of the following

 Click this option to filter events only if all the filters match.

Match Any of the following

 Click this option to filter events if any of the filters matches.

Live Output Stream

 Click Live Output Stream to expand or collapse this area. The output stream of the exploration is displayed in this area.

Properties

Click this button to view the properties of the output stream. You can add new constant properties. You can show/hide of the existing fields and reorder the fields in the results table.

Close

 Click Close when you are done with adding/viewing the properties.

Timestamp

 Click this button to add a timestamp to the output stream. This is a toggle button. Two types of time, Time (date) - normal time and Time (epoch) - the UNIX time, defined as the number of seconds lapsed since 00:00:00 Coordinated Universal Time (UTC), Thursday 1 January 1970.

Click this button again to remove the timestamps from the output stream.

Charts

 Click Charts to expand or collapse this area. The output stream is represented in a graphical format in this area.

1 Graph

 Default: 1 graph

Output stream data is represented as a single graph. When you click this button, the data is represented as only 1 graph.

2 Graphs

 Click this button to display the output stream data as 2 graphs.

3 Graphs

 Click this button to display the output stream data as 3 graphs.

Flip

 Click this button to flip from a horizontal side-by-side layout to a vertical (column) layout. This is a toggle button. Click this again to revert to the original layout.

Maximize

 Click this icon to maximize the window and view only graphs. Click this icon again to revert to the original view.
Windows drawer Click this icon to pull out the window drawer.

Range

 Select a value in the drop-down list and specify a numerical value. The range is defined on time units like: nanoseconds, microseconds, milliseconds, seconds, minutes, or hours.

  • Now - indicates that the range is as narrow as 1 nanosecond

  • Events - indicates that the range is the number of events

  • Unbounded - indicates that the range is not limited by any boundaries and the range since the server has started is considered.

When the range is anything other than Now, another field, Evaluation frequency appears.

Evaluation frequency

 Select a value in the drop-down list and specify a numerical. This value specifies how frequently the events are refreshed in the stream.

The evaluation frequency cannot be greater than the range.
Configure a Target Click this button to configure a target for the exploration.
Actions Click this button to view and select the available actions for an exploration.

Publish

 Click this button to publish the exploration. When more than one user tries to publish the exploration after modifications, a conflict arises. The following conflict resolution actions help you in resolving the conflict.

Overwrite

 This action overwrites the published version with the changes you have made.

Discard

 This action discards your and takes you to the latest published version.

Clone

 This action publishes the exploration under a different name, to capture changes from multiple users.

Unpublish

 Click this button to unpublish a published exploration.

Export

 Click this button to export the exploration.
Information Click this icon to view the details about the exploration like name, description, and tags.
Undo Changes Click this icon to undo the changes made to the exploration.
Redo Changes Click this icon to redo the changes made to the exploration.
