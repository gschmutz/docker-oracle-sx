Preferences - Catalog Page

Use this screen to configure the Catalog preferences.

Element Description

Default Sorting Column

 Select a value from the drop-down list. This value determines the column by which the items in the catalog are sorted.

Default Sorting Order

 Select if you want to sort the items in the catalog by ascending or descending order.

Default Page Size

 Select the number of items to be displayed on the catalog in a single page.
Restore To Default Click this button to restore the default settings.
Save Click this button to commit the changes made.
Cancel Click this button to abort the operation at any point of time.
