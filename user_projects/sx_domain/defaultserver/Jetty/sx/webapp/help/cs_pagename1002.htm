Configure a Target Dialog

Use this dialog to configure a target for the exploration.

Element Description

Destination Type

 Select a destination type.

CSV File

 If you select CSV File as the destination, you must enter information in the following fields.

File Name

 Enter a file name.

Append

 Select this option if you want to append the CSV file.
HTTP Publisher If you select HTTP Publisher as the destination, you must enter information in the following fields.

Server URL

 Enter the URL of the server.

Channel

 Enter the channel name.

Username

 Enter a user name.

Password

 Enter a password that authenticates the user name.
EDN If you select EDN as the destination, you must enter information in the following fields.

URL

 Enter a URL of EDN.

User Name

 Enter a user name.

Password

 Enter a password that authenticates the user name.

Get Event Definitions

 Click this button to retrieve the event definitions from the EDN Server.

Select Shape

 Select a shape from the drop-down list.

Target Property

 Indicates a field of the target shape.

Output Stream Property

 Indicates the exploration property that has been mapped to the target field. If you do not want to map, you can select it as No mapping.
JMS If you select JMS as the destination, you must enter information in the following fields.

URL

 Enter a URL of JMS.

User Name

 Enter the user name.

Password

 Enter a password that authenticates the user name.

JNDI name

 Enter the JNDI name of a topic or a queue.
REST If you select REST as the destination, you must enter information in the following fields.

Target URL

 Enter a target URL.
Cancel Click this button to abort the operation at any point of time.
Finish Click this button to finish and configure the target for the exploration.
