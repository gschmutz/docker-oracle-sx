Catalog Page

Use this page to view all the catalog items, explorations, streams, and references in the Stream Explorer.

Element Description

Explorations

 Default: selected

Select this button to display the explorations in the catalog. Deselect this button if you do not want to view the explorations in the catalog.

The exploration's status and its associated tags are displayed in the catalog.

  • Published - if the exploration is published

  • Draft - if the exploration is yet to be published.

Streams

 Default: selected

Select this button to display the streams in the catalog. Deselect this button if you do not want to view the streams in the catalog.

References

 Default: selected

Select this button to display the references in the catalog. Deselect this button if you do not want to view the references in the catalog.
Create New Item Select this button to create an exploration, stream, reference, or a pattern.
Mark item as favorite Click this button to mark the item as favorite.
Delete item Click this button to delete this item.

Sort By

 Use this drop-down list to sort the items on the catalog page.

Recent

 Select this option to view the items on the catalog page sorted by date (most recent being on the top).

Name

 Select this option to view the items on the catalog page in alphabetical order.

Favorite

 Select this option to view the items on the catalog page marked as favorites.
Show Click this option to select the number of items to be displayed on the Catalog page.
