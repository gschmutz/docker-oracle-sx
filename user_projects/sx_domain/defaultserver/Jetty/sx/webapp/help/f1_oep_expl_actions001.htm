Actions

Use this button to perform different actions on an exploration.

Element Description

Publish

 Click this button to publish the exploration.

Unpublish

 Click this button to unpublish the exploration.

Discard

 Click this button to discard any changes you have made in your draft copy and load the latest published state of the exploration.

Export

 Click this button to export the exploration to an external JAR file and download.

Export List

 This area displays the exploration details.

Resource

 This can be a source or the exploration itself. Both of them are available in the table as dependencies of the exploration. The exploration and its dependencies will be exported to a JAR file.

Type

 This is the type of the source or exploration itself. Both of them are available in the table as dependencies of the exploration.

Advanced Options

 This area displays the advanced options for the exploration.

Bundle Id

 Click this field to update the bundle details.

Bundle Name

 Click this field to update the bundle name.

Cancel

 Click this button to abort the operation at any point of time.

Export

 Click this button to export and save the exploration as a JAR file.
