Export Exploration

Use this dialog to export the exploration.

Element Description

Export List

 This area displays the exploration details.

Resource

 This is the name of the exploration and its dependencies to be exported.

Type

 This is the type of the resource (exploration/source).

Advanced Options

 This area displays the advanced options for the exploration.

Bundle ID

 Click this field to update the bundle details.

Bundle Name

 Click this field to update the bundle name.

Cancel

 Click this button to abort the operation at any point of time.

Export

 Click this button to export and save the exploration as a JAR file.
