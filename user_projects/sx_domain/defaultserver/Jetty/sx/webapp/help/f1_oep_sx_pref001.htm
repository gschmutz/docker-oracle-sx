Preferences - General Page

Use this screen to configure the General preferences.

Element Description
Start Page Select the page that appears as the start page. It can be either Welcome page or Catalog page.

Restore To Default

 Click this button to restore the default settings.
Save Click this button to commit the changes made.
Cancel Click this button to abort the operation at any point of time.
