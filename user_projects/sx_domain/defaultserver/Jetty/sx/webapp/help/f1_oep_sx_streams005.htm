Create Stream Wizard - Type Properties

Use this wizard to create a stream.

Element Description

Back

 Click this button to move to the previous screen of the wizard.
Type - CSV File Enter information in the following fields if you select the source as CSV File.

Source Path

 Enter a path for the source of the stream. Or, browse a file from you hard disk and upload that file.

Event Interval

 Enter a numeric value in this field. This is the time interval (in milliseconds) to wait between events.

Initial Delay

 Enter a numeric value in this field. This is the time interval (in milliseconds) to wait before reading the events.

Type - HTTP Subscriber

 Enter information in the following fields if you select the source as HTTP Subscriber.

Server URL

 Enter the URL of the server.

Channel Name

 Enter the channel name of the HTTP Subscriber.

Username

 Enter the user name.

Password

 Enter the password that authenticates the user name.

Type - EDN

 Enter information in the following fields if you select the source as EDN.

URL

 Enter the URL of the EDN.

Username

 Enter the username.

Password

 Enter the password that authenticates the user name.

Type - JMS

 Enter information in the following fields if you select the source as JMS.

URL

 Enter the URL of JMS.

Username

 Enter the username.

Password

 Enter the password that authenticates the user name.

JNDI Name

 Enter the JNDI name.

Type - REST

 Enter information in the following fields if you select the source as REST.

Context Path

 Enter a context path for the REST.
