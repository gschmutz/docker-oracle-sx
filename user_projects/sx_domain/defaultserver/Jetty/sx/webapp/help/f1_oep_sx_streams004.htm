Create Stream Wizard - Source Details

Use this wizard to create a stream.

Element Description

Name

 Name of the stream. Enter a name for the stream.

Description

 Description of the stream. Enter a description for the stream.

Tags

 Enter the tags with which you want to associate the stream.
Source Type Select a source type from the drop-down list for the stream.
Create Exploration with this source (Launch Exploration Editor) Select this option if you want to create an exploration with the source you have specified here for the stream. The Exploration Editor launches after the stream is created.
Cancel Click this button to abort the operation at any point of time.
Next Click this button to move to the next screen of the wizard.
